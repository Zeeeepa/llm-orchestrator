# @llm-dev-ops/llm-orchestrator-linux-x64

This package contains the native Linux x64 binary for LLM Orchestrator.

## Installation

This package is automatically installed as an optional dependency when you install the main `@llm-dev-ops/llm-orchestrator` package on a Linux x64 system. You typically don't need to install it directly.

```bash
npm install -g @llm-dev-ops/llm-orchestrator
```

## Platform Support

- **OS**: Linux
- **Architecture**: x64 (x86_64)
